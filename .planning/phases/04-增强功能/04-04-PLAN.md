# Plan: Frontend Client Management & Share Enhancements

## Objective

Create client management UI and enhance share link management with access logs and limits.

## Requirements

- **CLNT-01~04**: 客户管理 CRUD
- **PRIV-05**: 查看私密链接访问记录
- **PRIV-06**: 设置私密链接访问次数限制

## Tasks

### Task 1: Create Client API Client

Create `frontend/src/api/clients.ts`:

```typescript
export const clientsApi = {
  async getClients(),
  async createClient(data),
  async updateClient(id, data),
  async deleteClient(id),
  async getClientShares(id),
  async getClientAccessLogs(id),
}
```

### Task 2: Create Client Store

Create `frontend/src/stores/clients.ts`:

- Client list state
- CRUD actions
- Current client detail

### Task 3: Create Clients Management Page

Create `frontend/src/views/admin/Clients.vue`:

- Client list table
- Add/Edit client dialog
- Client detail shows:
  - Associated share links
  - Access history timeline

### Task 4: Update Shares Management Page

Update `frontend/src/views/admin/Shares.vue`:

- Add client selector when creating share
- Add maxAccess input field
- Show access log button for each share
- Access log dialog shows visit history

### Task 5: Create Access Log Component

Create `frontend/src/components/AccessLogDialog.vue`:

- Table of access records
- Filter by action type (view/download)
- Show IP and timestamp
