# Plan: Theme Switching & UI Polish

## Objective

Implement dark/light theme switching with localStorage persistence and polish the admin UI.

## Requirements

- **THEM-01**: 用户可以切换深色/浅色主题
- **THEM-02**: 主题选择在刷新后保持
- **THEM-03**: 管理后台界面清晰易用

## Tasks

### Task 1: Setup VueUse for Theme

Install and configure VueUse:

```bash
npm install @vueuse/core
```

Create `frontend/src/composables/useTheme.ts`:

```typescript
import { useDark, useToggle } from '@vueuse/core'

export const useTheme = () => {
  const isDark = useDark({
    storageKey: 'photo-show-theme',
    valueDark: 'dark',
    valueLight: 'light',
  })
  const toggleDark = useToggle(isDark)
  
  return { isDark, toggleDark }
}
```

### Task 2: Create CSS Variables for Themes

Create `frontend/src/styles/themes.css`:

```css
:root {
  --bg-primary: #ffffff;
  --bg-secondary: #f5f5f5;
  --bg-card: #ffffff;
  --text-primary: #1a1a1a;
  --text-secondary: #666666;
  --border-color: #e0e0e0;
  --accent-color: #409eff;
}

.dark {
  --bg-primary: #1a1a1a;
  --bg-secondary: #252525;
  --bg-card: #2a2a2a;
  --text-primary: #ffffff;
  --text-secondary: #aaaaaa;
  --border-color: #404040;
  --accent-color: #409eff;
}
```
