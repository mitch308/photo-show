# Plan: Frontend Batch Operations & Statistics

## Objective

Create frontend UI for batch operations (checkbox selection + batch action bar) and statistics display.

## Requirements

- **BATCH-01**: 管理员可以一次选择多个作品
- **BATCH-02**: 管理员可以批量修改作品的公开状态
- **BATCH-03**: 管理员可以批量移动作品到相册
- **BATCH-04**: 管理员可以批量删除作品
- **STAT-04**: 浏览和下载次数在管理后台可见

## Tasks

### Task 1: Create Batch API Client

Create `frontend/src/api/batch.ts`:

```typescript
export const batchApi = {
  async updateStatus(workIds: string[], isPublic: boolean),
  async moveToAlbum(workIds: string[], albumId: string | null),
  async deleteWorks(workIds: string[]),
}
```

### Task 2: Create Statistics API Client

Create `frontend/src/api/stats.ts`:

```typescript
export const statsApi = {
  async getWorksStats(),
  async getAlbumsStats(),
  async getOverview(),
}
```

### Task 3: Update Works Management Page

Update `frontend/src/views/admin/Works.vue`:

- Add checkbox column to table
- Track selected works in state
- Show batch action bar when works selected
- Batch action bar includes:
  - 批量公开/私密
  - 批量移动到相册
  - 批量删除
- Add view/download count columns to table

### Task 4: Create Batch Action Bar Component

Create `frontend/src/components/BatchActionBar.vue`:

- Show selected count
- Action buttons with confirmations
- Clear selection button
