## Key Decisions to Implement

- **D-02:** JWT stored in httpOnly Cookie (XSS protection)
- **D-03:** Access Token 7 days, Refresh Token 30 days
- **D-04:** Default admin account created on initialization
- **D-05:** Unified response format: `{ code, data, message }`
- **D-06:** HTTP status for request status, `code` field for business status

## Requirements Mapping

- **AUTH-01:** Login endpoint validates credentials and returns token
- **AUTH-02:** Token persists 7 days (httpOnly cookie survives browser close)
- **AUTH-03:** Logout adds token to Redis blacklist

## Pitfall Prevention

- Use bcrypt with proper salt rounds (10-12)
- Set secure cookie flags in production
- Redis TTL matches token expiration
