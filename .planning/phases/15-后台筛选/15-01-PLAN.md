       - 在 page-header 区域添加筛选控件（放在 h2 和 el-button 之间）：
         ```vue
         <div class="filters">
           <el-input
             v-model="searchQuery"
             placeholder="搜索作品标题..."
             clearable
             style="width: 200px"
           />
           <el-select v-model="statusFilter" placeholder="状态筛选" clearable style="width: 120px">
             <el-option label="公开" value="public" />
             <el-option label="私密" value="private" />
             <el-option label="置顶" value="pinned" />
           </el-select>
         </div>
         ```
       - 添加响应式变量：
         ```typescript
         const searchQuery = ref('');
         const statusFilter = ref('');
         ```
       - 实现 300ms 防抖搜索（参考 Clients.vue 模式）：
         ```typescript
         let searchTimeout: ReturnType<typeof setTimeout>;
         
         watch([searchQuery, statusFilter], () => {
           clearTimeout(searchTimeout);
           searchTimeout = setTimeout(() => {
             const filters: any = {};
             if (searchQuery.value) filters.title = searchQuery.value;
             if (statusFilter.value === 'public') filters.isPublic = true;
             if (statusFilter.value === 'private') filters.isPublic = false;
             if (statusFilter.value === 'pinned') filters.isPinned = true;
             worksStore.fetchWorks(filters);
           }, 300);
         });
         ```
       - 添加样式：
         ```css
         .filters {
           display: flex;
           gap: 12px;
           align-items: center;
         }
         ```
