# Plan 09-02: 前台跳转入口

**Requirement:** WORK-02 — 后台管理页面增加跳转前台画廊的入口

**Objective:** 在管理后台侧边栏添加"前台"链接，点击后在新标签页打开公开画廊首页，方便管理员快速预览前台效果。

---

## Context

**Why:** 管理员在后台管理作品时，经常需要预览前台展示效果。目前没有直接入口，需要手动输入地址或切换窗口。

**Key insight:** 纯前端 UI 增强，只需在 Dashboard.vue 侧边栏底部添加一个链接，样式与现有主题切换按钮一致。

---

## Tasks

### Task 1: 在侧边栏底部添加前台链接

<read_first>
- frontend/src/views/admin/Dashboard.vue — 当前侧边栏结构（约第 44-50 行）
</read_first>

<action>
在 `frontend/src/views/admin/Dashboard.vue` 的 `<template>` 部分，修改 `sidebar-footer` div（约第 44-50 行）：

将：
```vue
<div class="sidebar-footer">
  <button class="theme-toggle" @click="handleThemeToggle" :title="isDark ? '切换到浅色模式' : '切换到深色模式'">
    <span v-if="isDark" class="icon-sun">☀️</span>
    <span v-else class="icon-moon">🌙</span>
  </button>
  <button class="logout-btn" @click="handleLogout">退出登录</button>
</div>
```

改为：
```vue
<div class="sidebar-footer">
  <a href="/" target="_blank" class="gallery-link" title="在新窗口打开前台画廊">
    <span class="icon-gallery">🖼️</span>
  </a>
  <button class="theme-toggle" @click="handleThemeToggle" :title="isDark ? '切换到浅色模式' : '切换到深色模式'">
    <span v-if="isDark" class="icon-sun">☀️</span>
    <span v-else class="icon-moon">🌙</span>
  </button>
  <button class="logout-btn" @click="handleLogout">退出登录</button>
</div>
```

变更：在 theme-toggle 按钮前添加一个 `<a>` 链接
</action>

<acceptance_criteria>
- `frontend/src/views/admin/Dashboard.vue` 包含 `<a href="/" target="_blank" class="gallery-link"`
- 该链接位于 `sidebar-footer` div 内，`theme-toggle` 按钮之前
- 链接使用 `target="_blank"` 在新标签页打开
- 链接包含 `title="在新窗口打开前台画廊"` 属性
</acceptance_criteria>
