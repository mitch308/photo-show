# Plan: Log Rotation & Systemd Services

## Objective

Configure log rotation and create systemd service for application management.

## Tasks

### Task 1: Create Logrotate Configuration

Create `deploy/logrotate/photo-show`:

```
/var/log/photo-show/*.log {
    daily
    missingok
    rotate 30
    compress
    delaycompress
    notifempty
    create 0640 www-data www-data
    sharedscripts
    postrotate
        pm2 reloadLogs
    endscript
}
```

### Task 2: Create Logrotate Setup Script

Create `deploy/setup-logrotate.sh`:

```bash
#!/bin/bash

# Copy logrotate config
sudo cp deploy/logrotate/photo-show /etc/logrotate.d/photo-show

# Set permissions
sudo chmod 644 /etc/logrotate.d/photo-show

# Test configuration
sudo logrotate -d /etc/logrotate.d/photo-show

echo "Log rotation configured"
```

### Task 3: Create Deployment Script

Create `deploy/deploy.sh`:

```bash
#!/bin/bash
set -e

echo "=== Photo Show Deployment ==="

# Pull latest code
git pull origin main

# Backend
echo "Building backend..."
cd backend
npm install --production
npm run build

# Frontend
echo "Building frontend..."
cd ../frontend
npm install
npm run build

# Restart services

cd ..
pm2 reload photo-show-api

# Reload Nginx
```
