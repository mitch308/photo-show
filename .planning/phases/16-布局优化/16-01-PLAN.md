<task type="auto">
  <name>Task 1: 实现侧边栏导航独立滚动 (AUX-05)</name>
  <files>frontend/src/views/admin/Dashboard.vue</files>
  <read_first>
    - frontend/src/views/admin/Dashboard.vue (查看当前 .nav 样式)
  </read_first>
  <action>
    修改 Dashboard.vue 中的 `.nav` 样式，添加独立滚动能力。

    **当前代码 (第 86-89 行):**
    ```css
    .nav {
      flex: 1;
      padding: 20px 0;
    }
    ```

    **修改为:**
    ```css
    .nav {
      flex: 1;
      padding: 20px 0;
      overflow-y: auto;
      min-height: 0;
    }
    ```

    **说明:**
    - `overflow-y: auto` — 当导航项超出可视区域时显示滚动条
    - `min-height: 0` — 确保 flex 子元素可以收缩，允许滚动生效

    **原理:** 在 flex 容器中，子元素默认 min-height 为 auto，会阻止收缩。设置 min-height: 0 后，.nav 可以收缩到小于内容高度，从而启用滚动。Logo 和 sidebar-footer 保持固定位置。
  </action>
  <verify>
    <automated>grep -n "overflow-y: auto" frontend/src/views/admin/Dashboard.vue && grep -n "min-height: 0" frontend/src/views/admin/Dashboard.vue</automated>
  </verify>
  <acceptance_criteria>
    - Dashboard.vue 中 `.nav` 样式包含 `overflow-y: auto`
    - Dashboard.vue 中 `.nav` 样式包含 `min-height: 0`
    - 文件可以正常通过 Vue 编译（无语法错误）
  </acceptance_criteria>
  <done>
    - Dashboard.vue 已修改，`.nav` 样式添加了 `overflow-y: auto` 和 `min-height: 0`
    - 侧边栏导航区域可以独立滚动
    - Logo 和底部区域保持固定
  </done>
