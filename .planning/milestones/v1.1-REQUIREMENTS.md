# Requirements: v1.1 增强与修复

**Defined:** 2026-03-26
**Core Value:** 让摄影师轻松管理作品，让客户优雅地查看和获取作品

## v1.1 Requirements

v1.1 增强与修复需求，覆盖 Phases 7-12。

### Bug 修复

- [x] **BUG-01**: 水印功能集成 — 公开展示使用带水印缩略图，私密下载返回无水印原图
- [x] **BUG-02**: 下载功能修复 — 修复下载返回 JSON 的问题，确保返回源文件
- [x] **BUG-03**: 浏览量统计 — 公开画廊访问作品详情页时递增 viewCount

### 文件存储优化

- [x] **FILE-01**: Fast-MD5 预检查去重 — 上传前检查文件是否已存在，跳过重复上传
- [x] **FILE-02**: 智能缩略图生成 — 小图片不生成缩略图，避免浪费存储空间

### 作品信息增强

- [x] **WORK-01**: 文件信息展示 — 作品列表显示每个作品的总文件大小和文件数量
- [x] **WORK-02**: 前台跳转入口 — 后台管理页面有跳转前台画廊的入口链接

### 作品文件管理

- [x] **WORK-03**: 添加文件 — 为已有作品添加新的图片或视频文件
- [x] **WORK-04**: 删除文件 — 从作品中删除文件，最后一个文件需确认保护

### 工作室介绍

- [x] **STUD-01**: 基本信息配置 — 管理员在后台配置名称、Logo、联系方式
- [x] **STUD-02**: 富文本介绍 — 管理员使用富文本编辑器编辑工作室介绍内容
- [x] **STUD-03**: 前台展示 — 访客在前台查看工作室介绍页面

### 相册分享

- [x] **SHAR-01**: 相册分享入口 — 管理员可以为整个相册生成私密分享链接
- [x] **SHAR-02**: 相册分享页面 — 客户通过私密链接查看相册中的所有作品
- [x] **SHAR-03**: 相册分享下载 — 客户可以通过私密链接下载高清无水印原图
